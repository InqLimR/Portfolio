Добро пожаловать в репозиторий моего личного портфолио! Это центральный хаб, где собраны мои ключевые проекты, стек технологий и контактная информация. Проект создан с акцентом на чистый код, адаптивность и современный UX.

---

### 🚀 Технологический стек

*   **Frontend:** HTML5, CSS3 (Flexbox / Grid), JavaScript (ES6+)
*   **Бэкенд & Базы данных:** Firebase (Authentication, Firestore, Security Rules)
*   **Инструменты:** Git / GitHub, PyCharm, Font Awesome, Google Fonts

---

### 🎨 Ключевые фичи проекта

*   **Мультиязычность (Localization):** Полная поддержка переключения интерфейса между Русским и Английским языками (RU / EN).
*   **Кастомные темы (Theme Toggle):** Реализован бесшовный переключатель между Тёмной (Dark) и Светлой (Light) темами оформления.
*   **Адаптивный дизайн:** Интерфейс полностью оптимизирован под мобильные устройства, планшеты и ПК.

---

### 📂 Мои главные проекты в разработке

*   **💬 Kayla (ex InkMessenger):** Кастомный мессенджер нового поколения с авторизацией, приватными чатами и глубокой интеграцией правил безопасности Firebase Firestore.
*   **🚛 FLEUR Logistics:** Веб-портал и инфраструктура для виртуальной транспортной компании (VTC) в экосистеме TruckersMP.
